# SLACKER

> Don't worry, just slack!

## Installation

1. Clone this repo
1. In your terminal, enter `make` to compile
1. Run slacker by entering `./slacker`
1. Take a break without worrying about your status being `"away"` / `"offline"` :smirk: :smirk:
